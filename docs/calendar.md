## Google Calendar

1. Open Google Calendar on a desktop browser.
2. Next to **Other calendars**, select **+**.
3. Choose **From URL**.
4. Paste the HTTPS calendar URL above.
5. Select **Add calendar**.

Google controls refresh timing for subscribed calendars. For an immediate static import, download the `.ics` file and use **Settings → Import & export → Import**.

## Outlook

1. Open Outlook Calendar.
2. Select **Add calendar**.
3. Choose **Subscribe from web**.
4. Paste the HTTPS calendar URL.
5. Name it **Alberto IDR — 12 Week Sprint** and save.

Alternatively, download and open the `.ics` file to import a static copy.
